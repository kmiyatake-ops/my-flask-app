## 概要

本アプリケーションは、機械の日常点検・定期点検を効率的に行うためのチェックシートシステムです。画像上にチェックポイントを配置し、タップするだけでチェックを完了できます。チェック結果はCSV形式でエクスポートし、Google Sheetsなどのスプレッドシートに記録できます。

### 主な機能

- **画像ベースのチェックシート**: 機械の写真をアップロードして直感的に点検
- **吹き出し機能**: 画像上の任意の場所にチェックポイントを追加
- **ワンタップチェック**: タップ/クリックでチェック完了
- **モバイル対応**: スマートフォン・タブレットでも使用可能
- **データエクスポート**: CSV形式でチェック結果をエクスポート
- **履歴管理**: チェック日時を自動記録

## システム要件

### 必要な環境

- **OS**: Windows 10/11, macOS, Linux
- **Python**: 3.8 以上
- **ブラウザ**: Chrome, Firefox, Safari, Edge（最新版推奨）
- **インターネット接続**: Google Sheets連携時に必要

### 推奨環境

- **メモリ**: 4GB 以上
- **ストレージ**: 100MB 以上の空き容量
- **ディスプレイ**: 1024x768 以上の解像度

## インストール手順

### 1. Pythonの確認

まず、Pythonがインストールされているか確認します。

```bash
python --version
```

Python 3.8以上がインストールされている必要があります。インストールされていない場合は、[Python公式サイト](https://www.python.org/downloads/)からダウンロードしてインストールしてください。

アプリケーションフォルダで以下のコマンドを実行します。

※requirements.txtに記載されているパッケージは、Google Sheets API連携時に必要です。基本機能のみを使用する場合はFlaskのみで動作します。

1. コマンドプロンプト（Windows）またはターミナル（Mac/Linux）を開く

## 基本操作

### 1. 新規チェックシートの作成

#### ステップ1: トップページから作成開始

1. トップページの「新規チェックシート作成」ボタンをクリック
2. 作成ページに移動します

#### ステップ2: 基本情報の入力

1. **チェックシート名**: 例「機械A日常点検」「ボイラー定期点検」など
2. **機械画像**: 点検対象の機械の写真を選択
   - JPG、PNG形式対応
   - 推奨サイズ: 1920x1080ピクセル以下
   - ファイルサイズ: 5MB以下推奨

3. 画像プレビューで確認後、「作成して吹き出しを追加」ボタンをクリック

### 2. 吹き出し（チェックポイント）の追加

#### ステップ1: 追加モードの有効化

1. 「吹き出し追加モード」ボタンをクリック
2. ボタンが緑色になり、「追加モード: ON」と表示されます
3. 画像上にカーソルを合わせると十字カーソル（+）に変わります

#### ステップ2: チェックポイントの配置

1. 画像上のチェックしたい場所をクリック
2. ダイアログが表示されるので、チェック項目名を入力
   - 例: 「油圧計」「異常音」「漏れ確認」
3. OKをクリックすると吹き出しが追加されます

#### ステップ3: 複数ポイントの追加

必要な数だけステップ2を繰り返します。

#### ステップ4: 保存

1. 「保存」ボタンをクリック
2. 吹き出しの配置が保存されます

### 3. チェック実行

#### ステップ1: 追加モードの解除

1. 「吹き出し追加モード」ボタンを再度クリック
2. ボタンが青色に戻り、「吹き出し追加モード」と表示されます

#### ステップ2: チェック実行

1. 画像上の吹き出し（数字）をタップ/クリック
2. 吹き出しが緑色に変わり、チェックマーク（✓）が表示されます
3. チェック日時が自動記録されます

#### ステップ3: チェックの取り消し

チェック済みの吹き出しを再度タップ/クリックすると、チェックが取り消されます。

### 4. エクスポート

#### ステップ1: エクスポート実行

1. 「スプレッドシートへエクスポート」ボタンをクリック
2. CSVファイルが自動的にダウンロードされます
3. ファイル名: `checksheet_[ID]_[日付].csv`

#### ステップ2: Google Sheetsへのインポート

1. [Google Sheets](https://sheets.google.com)を開く
2. 「ファイル」→「インポート」を選択
3. 「アップロード」タブでダウンロードしたCSVファイルを選択
4. 以下の設定でインポート:
   - 「置換データ」: 「シートを置換」
   - 「区切り文字」: 「コンマを検出」
   - 「文字変換」: 「なし」
5. 「データをインポート」をクリック

## 機能詳細

### 吹き出しの編集・削除

#### 編集

1. 追加モードをONにする
2. 編集したい吹き出しをクリック
3. テキストを変更してOKをクリック

#### 削除

1. 画面下部の吹き出しリストから削除したい項目を探す
2. 「削除」ボタンをクリック
3. 確認ダイアログで「OK」をクリック

### チェックシート一覧

トップページでは、作成したすべてのチェックシートが一覧表示されます。

- **作成日順**: 新しい順に表示
- **カード形式**: チェックシート名と作成日を表示
- **クリックで遷移**: カードをクリックするとチェックシートページへ
- **削除機能**: 各カードの右上に削除ボタンがあります

### チェックシートの削除

不要になったチェックシートを削除できます。

#### 削除手順

1. トップページで削除したいチェックシートを探す
2. カード右上の「削除」ボタンをクリック
3. 確認ダイアログで「OK」をクリック
4. チェックシートと関連データが削除されます

#### 注意点

- 削除操作は取り消せません
- チェックシートに関連する以下のデータも削除されます:
  - アップロードされた画像ファイル
  - 吹き出し情報
  - チェック記録
- 誤って削除した場合、復元することはできません

### チェック記録の確認

チェックシートページでは、以下の情報が確認できます。

- **吹き出し番号**: 1から順番に割り振られます
- **チェック項目名**: 各吹き出しに設定されたテキスト
- **チェック状態**: 済/未
- **チェック日時**: チェックした日時

## トラブルシューティング

### アプリケーションが起動しない

**症状**: `python app.py` 実行時にエラーが発生する

**原因と解決策**:

1. **Flaskがインストールされていない**
   ```bash
   pip install flask
   ```

2. **Pythonのバージョンが古い**
   - Python 3.8以上をインストールしてください

3. **ポート5000が使用中**
   - 他のアプリケーションがポート5000を使用している可能性があります
   - app.pyの最後の行を変更してポートを変更できます:
     ```python
     app.run(debug=True, host='0.0.0.0', port=5001)
     ```

### 画像がアップロードできない

**症状**: 画像選択後にエラーが発生する

**原因と解決策**:

1. **ファイルサイズが大きすぎる**
   - 画像を圧縮してください（推奨: 5MB以下）

2. **ファイル形式が対応していない**
   - JPGまたはPNG形式を使用してください

3. **uploadsフォルダの権限問題**
   - フォルダが作成されているか確認してください
   - 手動でuploadsフォルダを作成してください

### 吹き出しが表示されない

**症状**: 保存した吹き出しが次回開いた時に消えている

**原因と解決策**:

1. **保存ボタンを押していない**
   - 吹き出し追加後に必ず「保存」ボタンをクリックしてください

2. **データベースファイルが破損**
   - `checksheet.db` ファイルを削除して、アプリを再起動してください
   - ※注意: これまでのデータが消去されます

### CSVエクスポートができない

**症状**: エクスポートボタンを押しても反応がない

**原因と解決策**:

1. **チェックシートが保存されていない**
   - まず吹き出しを保存してください

2. **ブラウザのポップアップブロック**
   - ポップアップを許可する設定にしてください

### モバイルで使いにくい

**症状**: スマートフォンで操作しづらい

**解決策**:

1. **画像をピンチイン/ピンチアウト**
   - 画像を拡大・縮小して操作できます

2. **横画面で使用**
   - 横画面にすると操作しやすくなります

## よくある質問

### Q1: 複数のユーザーで使用できますか？

A: 基本的には単一ユーザー向けです。複数ユーザーでの使用には、以下の対応が必要です:
- データベースを共有サーバーに移行
- ユーザー認証機能の追加
- Flaskアプリを本番環境（Heroku、AWS等）にデプロイ

### Q2: チェックシートを印刷できますか？

A: 直接の印刷機能はありませんが、以下の方法で印刷可能です:
1. ブラウザの印刷機能を使用（Ctrl+P）
2. CSVをExcel/Google Sheetsで開いて印刷

### Q3: 画像のサイズ制限はありますか？

A: 特定の制限はありませんが、推奨サイズは以下の通りです:
- 解像度: 1920x1080ピクセル以下
- ファイルサイズ: 5MB以下
大きすぎる画像は、読み込みや操作が遅くなる可能性があります。

### Q4: チェック履歴はどのくらい保存されますか？

A: データベースに保存されるため、基本的には無期限です。ただし、以下の点に注意してください:
- データベースファイル（checksheet.db）を削除すると履歴も消去されます
- 定期的にバックアップを取ることを推奨します

### Q5: オフラインで使用できますか？

A: はい、基本機能はオフラインで使用可能です。ただし、以下の機能はインターネット接続が必要です:
- Google Sheetsへのエクスポート・インポート
- 画像のアップロード（ローカルファイルであれば可能）

### Q6: セキュリティは大丈夫ですか？

A: 開発環境（debug=True）での使用を想定しています。本番環境での使用には、以下の対応が必要です:
- debugモードを無効化
- HTTPSの導入
- ユーザー認証の実装
- ファイアウォールの設定

### Q7: 他の言語に対応できますか？

A: 現在は日本語のみ対応しています。多言語対応には、以下の対応が必要です:
- テンプレートファイルの多言語化
- JavaScriptのメッセージ多言語化
- 翻訳ファイルの管理
