# 機械チェックシートアプリ

Flaskで作成された機械点検用チェックシートアプリケーションです。

## 機能

- **画像アップロード**: 機械の画像をアップロードしてチェックシートを作成
- **吹き出し追加**: 画像上にクリックで吹き出し（チェックポイント）を追加
- **タップでチェック**: 吹き出しをタップ/クリックしてチェック完了
- **レスポンシブデザイン**: 携帯・PCどちらでも使用可能
- **CSVエクスポート**: チェック結果をCSV形式でエクスポート（Google Sheetsにインポート可能）

## インストール

```bash
pip install -r requirements.txt
```

## 実行

```bash
python app.py
```

ブラウザで `http://localhost:5000` にアクセスしてください。

## 使い方

1. **新規チェックシート作成**
   - 「新規チェックシート作成」ボタンをクリック
   - チェックシート名を入力
   - 機械の画像をアップロード
   - 「作成して吹き出しを追加」ボタンをクリック

2. **吹き出しの追加**
   - 「吹き出し追加モード」ボタンをクリック
   - 画像上のチェックしたい場所をクリック
   - 吹き出しのテキストを入力
   - 必要な数だけ繰り返す
   - 「保存」ボタンをクリック

3. **チェック実行**
   - 追加モードをOFFにする
   - 吹き出しをタップ/クリックしてチェック完了
   - チェック済みの吹き出しは緑色になります

4. **エクスポート**
   - 「スプレッドシートへエクスポート」ボタンをクリック
   - CSVファイルがダウンロードされます
   - Google SheetsでCSVをインポートして使用可能

## Google Sheetsへのインポート方法

1. Google Sheetsを開く
2. 「ファイル」→「インポート」を選択
3. 「アップロード」タブでダウンロードしたCSVファイルを選択
4. インポート設定を確認して「データをインポート」をクリック

## 技術スタック

- Flask (Python Web Framework)
- SQLite (データベース)
- HTML/CSS/JavaScript (フロントエンド)
- レスポンシブデザイン (モバイル対応)
